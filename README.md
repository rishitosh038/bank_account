
## Overview
This project demonstrates **Object-Oriented Programming (OOP) concepts in Python** through a real-world **Bank Account Management System**.  
It uses classes, objects, encapsulation, inheritance, and method overriding to simulate basic banking operations.

This project is **interview-ready and production-structured**.

## How the Program Works
1. A base class `BankAccount` defines common properties and methods.
2. `SavingsAccount` and `CurrentAccount` inherit from `BankAccount`.
3. Encapsulation protects sensitive data like balance and account number.
4. Method overriding customizes behavior for different account types.
5. Multiple bank accounts are created to simulate real banking scenarios.
6. Operations such as deposit, withdrawal, and interest calculation are performed.
